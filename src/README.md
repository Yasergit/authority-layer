# Workspace Source Map

AuthorityLayer publishes its implementation from `packages/core/src/`.

This top-level `src/` directory exists as a contributor-facing source map so the repository has an obvious entrypoint when browsing the workspace at a glance.

Primary package entrypoints:

- `packages/core/src/index.ts`
- `packages/core/src/AuthorityLayer.ts`
- `packages/core/src/EnforcementHalt.ts`
- `packages/core/src/enforcement/`
- `packages/core/src/cli/`
